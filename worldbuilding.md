# 《我的女儿是财神》世界观设定

## 天庭财神体系

### 小七的身份
- **天庭官职：** 财神（正七品），负责人间财运调配
- **任期：** 已任职300年，是天庭最年轻的财神
- **职责：** 审批人间大额财运变动（暴富/破产/国运级经济波动）
- **业绩：** 点拨过范蠡（陶朱公）、沈万三、胡雪岩等历史富豪
- **性格特点：** 三百年看透人间财富起落，觉得"人间的钱最好玩"，对凡人既傲娇又心软

### 下凡规则
- 天上一日 = 地上一年
- 小七偷跑了100天 = 地上100年（但实际上她只会在人间待到肉身成年，约18年）
- 下凡时法力被封印90%，只留10%的本源能力
- 封印解除进度：按年龄/经历/心境逐步开放
  - 婴儿期（0-3岁）：鉴宝之眼（被动），能看到物品价值
  - 幼童期（3-6岁）：财运感知，能看人/看公司的财运颜色
  - 少年期（7-14岁）：投资直觉，能感知最佳投资时机
  - 青年期（15-18岁）：封印大面积解除，接近满级
  - 18岁生日：天庭追兵到来，必须做出选择
- **违反规则的代价：** 被打入轮回，三百年记忆和修为全部归零

### 财运颜色系统（小七的独有视角）
- **璀璨金：** 绝世财富，百年一遇的暴富机会
- **明黄色：** 稳定增长，好的投资标的
- **淡金色：** 小赚，稳赚不赔但不多
- **灰白色：** 不赚不赔，平庸
- **暗灰色：** 小亏，需要避开
- **黑色：** 大亏/破产/灾难
- **血红：** 血光之灾，伴随人身危险
- **混沌色：** 看不清——意味着巨大的不确定性（可能暴富也可能血本无归）

## 人间搞钱体系（按阶段解锁）

### 第一阶段：古玩捡漏（0-3岁，婴儿期）
- 场景：旧货市场、典当行、拍卖会、乡下收货
- 手法：小七看中的东西 → 妈妈犹豫但买下 → 价值惊人
- 经典场景：
  - 地摊50块买的碗 → 明宣德青花 → 拍卖800万
  - 废品站捡的破画 → 唐伯虎真迹 → 无价
  - 老农手里收的玉佩 → 战国和田白玉 → 天价
  - 乡下猪圈里翻出的铜器 → 商周青铜 → 国宝

### 第二阶段：赌石暴富（3-5岁，幼童期）
- 场景：云南/缅甸翡翠公盘、翡翠市场
- 手法：小七指哪块 → 妈妈照买 → 切出极品
- 经典场景：
  - 别人不要的废料 → 切出帝王绿
  - 一万块的石头 → 切出满绿翡翠 → 价值千万
  - 以小博大，从几十万滚到几个亿
  - 打脸赌石圈的"老行家"

### 第三阶段：投资封神（5-10岁）
- 场景：股市、风投、收购不良资产
- 手法：小七看公司的"财运颜色" → 妈妈精准投资
- 经典场景：
  - 别人看烂的ST股 → 三个月后重组暴涨10倍
  - 投资被所有人不看好的初创公司 → 成为独角兽
  - 在楼市最低谷买入核心地段
  - 精准做空竞争对手的股票

### 第四阶段：商业帝国（10-18岁）
- 场景：商场、资本市场、全球扩张
- 手法：女儿+妈妈联手，建立多元化商业帝国
- 领域：地产、科技、金融、文化、新能源
- 经典场景：
  - 收编前夫的顾氏集团（用收购的方式）
  - 打败国际资本做空
  - 建立全球最大的私人基金会
  - 最终登顶世界首富

### 第五阶段：终极选择（18岁）
- 天庭追兵到来
- 小七面临选择：回天庭继续做财神，还是留在人间当林若溪的女儿
- 最终结局待定（开放式/HE）

## 天庭追兵设定（后期反派）
- **天庭执法使：** 冷面无私，但不是纯反派——有自己的理解
- **冲突核心：** 小七的行为扰乱了人间财运秩序，但她做的事本质上是"帮助好人"，天庭规则是否合理？
- **结局方向：** 小七用行动证明"财运不应只由天定，人的努力也很重要"，推动天庭改革
